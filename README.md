#  Impulse Buying Behavior Analytics Dashboard

##  Overview
This project analyzes consumer transaction data to detect and visualize impulse buying behavior using Power BI.

## Objectives
- Identify impulsive purchase patterns
- Analyze late-night spending behavior
- Evaluate discount-driven purchases
- Generate behavioral spending insights

##  Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
